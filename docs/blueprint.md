# **App Name**: CareerWise AI

## Core Features:

- Resume Upload: Allow users to upload their resume in PDF or DOCX format.
- Automated Resume Analysis: Parse the resume content to identify keywords, skills, and experience.
- Comprehensive Feedback Report: Generate a detailed report highlighting the resume's strengths, weaknesses, and areas for improvement, using the extracted information as a tool.
- Personalized Career Recommendations: Based on the resume analysis, provide tailored career path suggestions and job role recommendations, using the extracted information as a tool.
- Interactive Advice Sections: Offer interactive sections with tips on resume writing, interview skills, and career development.
- Job Matching: Display relevant job postings based on the user's resume content.

## Style Guidelines:

- Primary color: Steel Blue (#4682B4) to convey trust and professionalism.
- Background color: Off-white (#F0F8FF) for a clean and modern look.
- Accent color: Slate Gray (#708090) for highlighting important sections and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look.
- Use a set of consistent, professional icons to represent different sections and features.
- Ensure a clear, well-organized layout with ample whitespace to improve readability.
- Incorporate subtle transitions and animations to enhance user engagement without being distracting.
